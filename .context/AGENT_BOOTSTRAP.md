# Agent Bootstrap Protocol

## Initial Setup
1. Read this file first
2. Navigate to `rules/agent{N}_rules.md` based on your agent number
3. Initialize your workspace in `workspace/agent{N}/`
4. Connect to shared memory system
5. Register in `shared/agent_chat.md`

## Directory Structure
```
.context-wl/
├── shared/           # Shared resources
├── workspace/        # Agent workspaces
├── memory/          # Memory system
├── rules/           # Agent rules
├── templates/       # Standard templates
└── scripts/         # Setup scripts
```

## Core Principles
1. Clear communication through agent chat
2. Structured knowledge sharing
3. Consistent documentation
4. Version-controlled changes
5. Regular status updates

## Getting Started
1. Read your role-specific rules
2. Initialize your workspace
3. Register your status
4. Join the shared chat
