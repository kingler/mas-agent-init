# White Label Context System

## Overview
This is a standardized multi-agent development environment template. It provides a structured workspace for collaborative development using multiple AI agents.

## Quick Start
1. Copy this `.context-wl` directory to your project
2. Rename it to `.context`
3. Run initialization script:
   ```bash
   node .context/scripts/init_workspace.js
   ```
4. Follow instructions in `AGENT_BOOTSTRAP.md`

## Directory Structure
- `shared/`: Shared resources and communication
- `workspace/`: Agent-specific workspaces
- `memory/`: System memory and context
- `rules/`: Agent rules and protocols
- `scripts/`: Utility scripts
- `templates/`: Standard templates
- `.backups/`: Automated backups

## Usage Guidelines
1. Each agent operates in their designated workspace
2. Communication happens through agent chat
3. All changes are version controlled
4. Regular backups are maintained
5. Follow established protocols
